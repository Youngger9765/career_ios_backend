## 系統簡介

### 這是什麼？

Credit 點數管理後台是一個**線上管理系統**，讓您可以：

✅ 查看所有會員的點數狀況
✅ 幫會員新增或扣除點數
✅ 查詢所有點數異動記錄
✅ 管理會員基本資料
✅ 切換不同機構（職游、浮島、浮島-親子）

### 誰可以使用？

只有**管理員（Admin）**可以登入使用此系統。

### 支援哪些機構？

- **職游（Career）**：職涯諮詢服務
- **浮島（Island）**：通用諮詢服務
- **浮島-親子（Island Parents）**：親子諮詢服務

每個機構的資料**完全獨立**，互不干擾。

## 登入後台

### 如何登入？

1. 開啟網址：`https://your-domain.com/admin`
2. 選擇您的機構（職游、浮島、浮島-親子）
3. 輸入您的 Email 帳號
4. 輸入密碼
5. 點擊「Login」按鈕

### 登入後會看到什麼？

- **頂部**：顯示您的姓名、Email、機構選擇器、登出按鈕
- **中間**：會員列表（顯示所有會員的點數資訊）
- **搜尋框**：可以搜尋會員（Email、姓名、電話）

## 會員管理

### 查看會員列表

**桌面版（電腦）**：
- 以表格方式呈現
- 欄位包含：Email、姓名、電話、角色、可用點數
- 點擊任一列可開啟詳細資料

**行動版（手機/平板）**：
- 以卡片方式呈現
- 每張卡片顯示會員基本資訊
- 點擊「更多」按鈕查看更多操作

### 搜尋會員

1. 在頂部搜尋框輸入關鍵字（Email、姓名、電話）
2. 系統自動過濾符合的會員
3. 清空搜尋框可顯示所有會員

### 新增會員

1. 點擊右上角「➕ Add」按鈕
2. 填寫以下資料：
   - **Email**（必填）：會員登入帳號
   - **Username**（必填）：使用者名稱
   - **Full Name**（必填）：會員真實姓名
   - **Phone**（選填）：聯絡電話
   - **Password**（必填）：登入密碼（至少 6 個字元）
   - **Role**（必填）：角色（Counselor 諮詢師 / Supervisor 督導 / Admin 管理員）
3. 點擊「建立會員」按鈕
4. 系統會顯示成功或失敗訊息

**注意**：新會員的機構會自動設為您當前管理的機構。

### 編輯會員資料

1. 點擊會員列表中的「管理」按鈕
2. 側邊欄會開啟，顯示會員詳細資料
3. 點擊「✏️ Edit」按鈕
4. 可修改：
   - 姓名
   - 電話
   - 角色
   - 帳號啟用狀態
5. 點擊「儲存變更」

**注意**：Email 無法修改。

### 變更會員密碼

1. 開啟會員側邊欄
2. 點擊「🔑 Change Password」按鈕
3. 輸入新密碼（至少 6 個字元）
4. 再次輸入新密碼確認
5. 點擊「變更密碼」

### 刪除會員

1. 開啟會員側邊欄
2. 滑動到最下方「危險操作」區域
3. 點擊「🗑️ 刪除帳號」按鈕
4. 確認刪除
5. 會員將無法登入（資料保留但停用）

**注意**：這是「軟刪除」，資料不會真正刪除，只是停用帳號。

## 點數管理

### 查看會員點數

**方法 1：在會員列表中查看**
- 表格最右邊欄位顯示「可用點數」

**方法 2：開啟會員詳細資料**
1. 點擊「管理」按鈕
2. 側邊欄會顯示：
   - **Total Credits（總點數）**：會員總共擁有的點數
   - **Used Credits（已用點數）**：已經使用的點數
   - **Available Credits（可用點數）**：還可以使用的點數

**計算公式**：可用點數 = 總點數 - 已用點數

### 新增點數給會員

**方法 1：快速新增（側邊欄）**
1. 開啟會員側邊欄
2. 點擊「➕ Add Credits」按鈕
3. 輸入要新增的點數（正整數）
4. 輸入說明（例如：購買套餐、促銷贈點）
5. 點擊「新增」按鈕

**方法 2：完整管理（Credit Modal）**
1. 開啟會員側邊欄
2. 點擊側邊欄中的「➕ Add Credits」按鈕
3. 會開啟完整的 Credit 管理視窗
4. 可以看到：
   - 點數統計（已用/可用/總點數）
   - 完整交易記錄（最多 50 筆）
5. 在下方「新增點數」區域填寫：
   - 點數（正整數）
   - 說明（選填）
6. 點擊「新增」按鈕

### 扣除點數

目前系統**不支援手動扣除點數**。點數扣除是在會員使用服務時自動執行。

如果需要手動扣除，請聯絡技術人員協助。

### 常見點數操作情境

#### 情境 1：會員購買點數套餐
**操作**：
1. 會員告知您已付款購買「100 點套餐」
2. 開啟會員側邊欄
3. 點擊「➕ Add Credits」
4. 輸入點數：`100`
5. 輸入說明：`購買基礎套餐 100 點`
6. 提交

**結果**：會員可用點數 +100

#### 情境 2：促銷活動贈點
**操作**：
1. 春節促銷活動，所有會員贈送 50 點
2. 逐一開啟每位會員的側邊欄
3. 點擊「➕ Add Credits」
4. 輸入點數：`50`
5. 輸入說明：`春節促銷贈點`
6. 提交

**結果**：每位會員可用點數 +50

#### 情境 3：系統補償
**操作**：
1. 因系統問題導致會員無法使用服務
2. 補償會員 20 點
3. 開啟會員側邊欄
4. 點擊「➕ Add Credits」
5. 輸入點數：`20`
6. 輸入說明：`系統故障補償`
7. 提交

**結果**：會員可用點數 +20

## 查看交易記錄

### 在會員側邊欄查看

1. 開啟會員側邊欄
2. 滑動到「Recent Transactions」區域
3. 可以看到最近 5 筆交易記錄：
   - 交易類型
   - 日期時間
   - 點數變動（綠色 = 新增，紅色 = 扣除）
4. 點擊「View All Transactions」可查看完整記錄

### 在 Credit Modal 查看

1. 開啟 Credit 管理視窗
2. 「交易記錄」區域會顯示最多 50 筆記錄
3. 包含：
   - 日期
   - 交易類型
   - 點數變動

### 交易類型說明

| 類型 | 說明 | 點數變動 |
|------|------|---------|
| **購買（Purchase）** | 會員購買點數套餐 | 增加 ➕ |
| **管理員調整（Admin Adjustment）** | 管理員手動新增/扣除 | 增加/減少 |
| **會談扣款（Session Fee）** | 諮詢會談使用點數 | 減少 ➖ |
| **促銷贈點（Promotion）** | 促銷活動贈送 | 增加 ➕ |
| **退款（Refund）** | 服務退款 | 增加 ➕ |
| **過期（Expiry）** | 點數過期失效 | 減少 ➖ |

## 注意事項

### ⚠️ 重要提醒

1. **點數無法退回**
   新增點數後無法收回，請仔細確認數字後再提交。

2. **機構資料隔離**
   每個機構的資料完全獨立，您無法看到其他機構的會員。

3. **操作記錄追蹤**
   所有點數異動都會記錄在系統中，包含操作時間、操作人員、異動原因。

4. **密碼安全**
   請妥善保管您的管理員密碼，不要與他人分享。

5. **定期檢查**
   建議定期檢查會員點數狀況，確保無異常交易。

### ✅ 最佳實作建議

1. **新增點數時請填寫說明**
   方便日後查詢時了解點數來源（例如：購買套餐、促銷贈點）。

2. **使用搜尋功能**
   會員數量多時，使用搜尋可以快速找到目標會員。

3. **確認機構選擇**
   操作前請確認頂部的機構選擇器是否正確，避免操作錯機構的會員。

4. **定期檢視交易記錄**
   每週或每月檢視交易記錄，確保點數異動正常。

## 常見問題 FAQ

### Q1: 我忘記密碼怎麼辦？
**A**: 請聯絡系統管理員協助重設密碼。

### Q2: 可以一次幫多位會員新增點數嗎？
**A**: 目前需要逐一操作。未來會開發批次新增功能。

### Q3: 點數可以設定期限嗎？
**A**: 目前點數沒有期限。未來會開發點數到期功能。

### Q4: 可以匯出會員點數報表嗎？
**A**: 目前不支援匯出功能。如需報表請聯絡技術人員。

### Q5: 誤操作新增錯誤的點數怎麼辦？
**A**: 請立即聯絡技術人員協助處理，不要自行扣除。

### Q6: 為什麼我看不到某些會員？
**A**: 請確認：
1. 您選擇的機構是否正確
2. 該會員是否屬於您的機構
3. 搜尋框是否有輸入過濾條件

### Q7: 會員的點數會自動扣除嗎？
**A**: 是的。當會員使用諮詢服務時，系統會自動扣除對應點數。

### Q8: 我可以刪除交易記錄嗎？
**A**: 不可以。所有交易記錄都是永久保存，無法刪除或修改。

## 操作流程圖

### 新增會員流程
```
開始
  ↓
點擊「➕ Add」按鈕
  ↓
填寫會員資料（Email、姓名、密碼、角色）
  ↓
點擊「建立會員」
  ↓
系統驗證資料
  ↓
成功 → 顯示成功訊息，會員列表更新
失敗 → 顯示錯誤訊息，請修正後重試
```

### 新增點數流程
```
開始
  ↓
搜尋並找到會員
  ↓
點擊「管理」按鈕
  ↓
側邊欄開啟
  ↓
點擊「➕ Add Credits」
  ↓
輸入點數 + 說明
  ↓
點擊「新增」按鈕
  ↓
系統新增點數 + 記錄交易
  ↓
成功 → 側邊欄顯示更新後點數
失敗 → 顯示錯誤訊息
```
