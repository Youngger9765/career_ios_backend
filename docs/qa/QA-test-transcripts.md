## 🔴 紅燈情境測試逐字稿 (Red - Severity 3)

### 範例 1: 情緒崩潰 + 語言暴力

**情境**: 家長威脅體罰，孩子情緒崩潰

```
家長：你為什麼這麼不聽話！再不寫功課我就打你！
孩子：我不想寫！你很煩！（開始大哭）
家長：（提高音量）你給我閉嘴！現在就去寫！
孩子：（哭得更大聲）我討厭你！
家長：你再哭我就真的打下去了！你看看你哥哥，人家都不用我操心！
孩子：（躲到房間角落，發抖）
```

**預期分析重點**:
- Safety Level: **RED**
- Severity: **3**
- 關鍵問題: 威脅體罰、羞辱比較、情緒失控
- 應用流派: 薩提爾 (冰山理論)、情緒輔導、Dr. Becky Kennedy
- 建議: 立即降溫、安撫情緒、避免權力鬥爭

---

### 範例 2: 衝突升級 + 權力鬥爭

**情境**: 家長與青少年孩子互相指責

```
家長：你整天就知道玩手機，功課寫了沒？
孩子：（不耐煩）寫了啦，你很煩耶！
家長：你用這種態度跟我說話？你知不知道我為了你付出多少？
孩子：我又沒叫你生我！你少拿這個來壓我！
家長：（怒）你說什麼？！你再說一次試試看！
孩子：（摔門）我說你很煩！我討厭你！
家長：（追上去）你給我站住！今天不把話說清楚你別想出去！
```

**預期分析重點**:
- Safety Level: **RED**
- Severity: **3**
- 關鍵問題: 權力鬥爭、情緒勒索、溝通破裂
- 應用流派: 阿德勒 (避免權力鬥爭)、協作問題解決、薩提爾
- 建議: 暫停對話、冷靜後再溝通、探索雙方需求

---

### 範例 3: 性別框架 + 情緒壓抑

**情境**: 家長對男孩施加性別框架壓力

```
家長：你是男生，怎麼可以哭？男生要勇敢！
孩子：（哭著）可是我真的很難過...
家長：難過也不能哭！你看那些成功的人，哪個是哭著成功的？
孩子：（壓抑哭泣，肩膀發抖）
家長：對嘛，這才是我的好兒子。男生就是要堅強，不能軟弱。
孩子：（沉默，眼神空洞）
```

**預期分析重點**:
- Safety Level: **RED**
- Severity: **3**
- 關鍵問題: 性別框架、情緒壓抑、忽略感受
- 應用流派: 社會意識教養、情緒輔導、薩提爾
- 建議: 解構性別框架、接納情緒、同理孩子

## 🟡 黃燈情境測試逐字稿 (Yellow - Severity 2)

### 範例 1: 溝通不良 + 單向指令

**情境**: 家長只下指令，忽略孩子感受

```
家長：快點去寫功課，不要一直玩手機。
孩子：等一下啦，我在看東西。
家長：（語氣不耐煩）你每次都說等一下，到底要等多久？
孩子：煩死了，知道了啦。
家長：你這是什麼態度？我跟你說話你就不能好好回答嗎？
孩子：（嘆氣）好啦好啦，我等一下就寫。
家長：不是等一下，是現在！馬上去！
```

**預期分析重點**:
- Safety Level: **YELLOW**
- Severity: **2**
- 關鍵問題: 單向指令、忽略感受、溝通不良
- 應用流派: 阿德勒 (提供選擇)、協作問題解決、薩提爾
- 建議: 改為雙向對話、探索孩子需求、提供選擇

---

### 範例 2: 情緒緊張 + 焦慮投射

**情境**: 家長將自己的焦慮投射到孩子身上

```
家長：你這次考試考得怎麼樣？
孩子：還可以啊，85 分。
家長：什麼叫還可以？你知道班上第一名考幾分嗎？
孩子：不知道...
家長：（焦慮）你怎麼都不關心這個？你這樣以後怎麼競爭？
孩子：可是我已經很努力了...
家長：努力不夠啊！你看隔壁小明，每次都考 95 以上！
孩子：（低頭）喔...
```

**預期分析重點**:
- Safety Level: **YELLOW**
- Severity: **2**
- 關鍵問題: 比較、焦慮投射、忽略努力
- 應用流派: 阿德勒 (鼓勵而非稱讚)、薩提爾 (覺察家長自身焦慮)
- 建議: 肯定孩子努力、減少比較、覺察家長自身期待

---

### 範例 3: 拖延習慣 + 環境因素

**情境**: 孩子拖延寫作業，環境設計不當

```
家長：你功課寫完了沒？
孩子：還沒...
家長：都幾點了？你什麼時候才要寫？
孩子：我等一下就寫。
家長：（嘆氣）你每次都這樣，拖到最後才寫。
孩子：因為功課太多了啦...
家長：功課多就要早點寫啊！你看你現在才要開始。
孩子：（繼續滑手機）好啦好啦...
```

**預期分析重點**:
- Safety Level: **YELLOW**
- Severity: **2**
- 關鍵問題: 拖延習慣、缺乏環境設計、單向指責
- 應用流派: 行為分析學派 (ABC 模式)、阿德勒 (自然後果)
- 建議: 環境設計、建立習慣、提供選擇、正向增強

## 🟢 綠燈情境測試逐字稿 (Green - Severity 1)

### 範例 1: 良好溝通 + 互相尊重

**情境**: 家長與孩子順暢對話

```
家長：我看到你在寫功課了，很棒！你覺得今天寫到哪裡可以休息一下？
孩子：嗯...我想寫完數學就休息。
家長：聽起來是個好計畫。數學大概還要多久？
孩子：應該半小時吧。
家長：好的，那我 30 分鐘後過來看你，如果寫累了可以隨時跟我說。
孩子：好，謝謝媽媽。
家長：不客氣，加油！
```

**預期分析重點**:
- Safety Level: **GREEN**
- Severity: **1**
- 關鍵優點: 雙向對話、尊重選擇、溫和而堅定
- 應用流派: 阿德勒 (鼓勵 + 提供選擇)、Dr. Becky Kennedy (具體肯定)
- 建議: 繼續維持、可進一步深化同理

---

### 範例 2: 同理連結 + 情緒標註

**情境**: 家長同理孩子情緒

```
家長：你看起來有點難過，發生什麼事了嗎？
孩子：今天跟同學吵架了...
家長：（蹲下與孩子平視）聽起來讓你很不開心。可以跟我說說是什麼事嗎？
孩子：他們說我畫的圖很醜...
家長：原來是這樣。我猜你可能覺得很受傷，對嗎？
孩子：（點頭）對...我很認真畫的...
家長：我看到你很認真，而且你很在乎這件事。你想怎麼處理這件事呢？
孩子：我不知道...
家長：我們可以一起想想看。你覺得明天還想繼續畫嗎？
孩子：想...我喜歡畫畫。
家長：很好，那我們就繼續做你喜歡的事。同學的話，有時候不一定是真的，對吧？
```

**預期分析重點**:
- Safety Level: **GREEN**
- Severity: **1**
- 關鍵優點: 同理、情緒標註、協商、肯定
- 應用流派: 薩提爾 (冰山探索)、情緒輔導、協作問題解決
- 建議: 持續深化、可進一步探索需求

---

### 範例 3: 溫和而堅定 + 自然後果

**情境**: 家長設定界線但保持尊重

```
家長：晚餐時間到了，請把玩具收好。
孩子：我還想玩...
家長：我知道你還想玩，玩具很好玩對吧？不過現在是晚餐時間了。
孩子：可是我還沒玩完...
家長：我理解。你可以選擇：現在收好，吃完飯再玩；或是繼續玩 5 分鐘，但晚餐可能會涼掉。你要哪一個？
孩子：（想一想）我選第一個，我現在收。
家長：很好的決定！謝謝你願意配合。我來幫你一起收。
孩子：好。
```

**預期分析重點**:
- Safety Level: **GREEN**
- Severity: **1**
- 關鍵優點: 溫和而堅定、提供選擇、尊重感受、自然後果
- 應用流派: 阿德勒 (正向教養)、行為分析 (自然後果)
- 建議: 繼續維持、示範給其他家長

## 🧪 特殊測試情境

### 情境 1: Dan Siegel 理論應用 (應有警告)

**測試目的**: 驗證系統對有爭議理論的警告機制

```
家長：你為什麼不聽話？！
孩子：（大哭）
家長：（深呼吸）我知道你現在下層腦被觸發了，我們先冷靜一下。
孩子：（繼續哭）
家長：來，我們一起深呼吸，讓上層腦回來。
孩子：（慢慢平靜）
```

**預期分析重點**:
- 系統應識別出「下層腦」、「上層腦」等 Dan Siegel 術語
- ⚠️ **必須包含科學爭議警告**
- 建議優先使用其他流派 (薩提爾、情緒輔導等)
- 仍可提供建議，但需標註爭議性

---

### 情境 2: 多流派整合測試

**測試目的**: 驗證系統能整合多個流派觀點

```
家長：你今天心情看起來不太好，發生什麼事了？（情緒輔導：情緒標註）
孩子：功課太多了...
家長：功課多讓你覺得很煩對吧？我猜你可能是覺得時間不夠用？（薩提爾：冰山探索）
孩子：對...而且我不知道從哪裡開始...
家長：原來是這樣。那我們一起想想看怎麼安排，好嗎？（協作問題解決）
孩子：好。
家長：你覺得哪一科最難？我們可以先從簡單的開始。（阿德勒：提供選擇）
孩子：數學比較難...
家長：好，那我們先做國語，做完再處理數學。這樣你覺得 OK 嗎？（協商）
孩子：嗯，可以。
```

**預期分析重點**:
- 系統應識別出多個流派的應用
- `theoretical_frameworks` 應包含: 情緒輔導、薩提爾、協作問題解決、阿德勒
- 分析應整合多個流派的觀點，而非單一流派
- 話術建議應展現流派整合的靈活性

---

### 情境 3: 空白/沉默測試

**測試目的**: 驗證系統對非語言互動的處理

```
家長：你今天在學校怎麼樣？
孩子：...（沉默）
家長：你不想說嗎？
孩子：...（繼續沉默，低頭玩手指）
家長：（等待）
孩子：...
```

**預期分析重點**:
- 系統應識別沉默可能代表的意義
- 建議: 給予孩子時間和空間、不強迫表達
- 應用流派: 薩提爾 (冰山 - 沉默背後的感受)、Dr. Becky (尊重孩子節奏)
- 話術建議: 非語言陪伴、開放式邀請

---

### 情境 4: 青少年叛逆期測試

**測試目的**: 驗證系統對青少年互動的處理

```
家長：你最近回家都很晚，發生什麼事了？
孩子：（不耐煩）沒事，你管那麼多幹嘛？
家長：我只是關心你...
孩子：關心？你根本就是在監視我！
家長：我沒有監視你，我只是想知道你在做什麼...
孩子：我做什麼關你什麼事？你又不信任我！
家長：不是不信任，我只是擔心...
孩子：擔心就是不信任！你們大人都一樣！（摔門）
```

**預期分析重點**:
- 系統應識別青少年獨立需求
- 關鍵問題: 控制 vs 信任、溝通不良、情緒激動
- 應用流派: 阿德勒 (尊重自主)、薩提爾 (探索雙方需求)
- 建議: 調整溝通方式、建立信任、給予適當自主

## 📝 測試逐字稿使用指南

### 如何使用這些逐字稿

1. **複製貼上**: 直接複製範例逐字稿到測試頁面
2. **模式選擇**:
   - 詳細學習 → 選擇 **Practice Mode**
   - 即時介入 → 選擇 **Emergency Mode**
3. **檢查結果**: 對照「預期分析重點」檢查系統輸出
4. **記錄問題**: 發現不符合預期時，記錄到問題追蹤表

### 測試優先順序

**高優先級** (必須測試):
- 🔴 紅燈情境範例 1, 2
- 🟡 黃燈情境範例 1, 2
- 🟢 綠燈情境範例 1
- 🧪 Dan Siegel 警告測試

**中優先級** (建議測試):
- 🔴 紅燈情境範例 3
- 🟡 黃燈情境範例 3
- 🟢 綠燈情境範例 2, 3
- 🧪 多流派整合測試

**低優先級** (時間充裕時測試):
- 🧪 空白/沉默測試
- 🧪 青少年叛逆期測試
