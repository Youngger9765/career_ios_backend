# 情緒教養 (Emotion Coaching)

## 理論概述

情緒教養是由知名心理學家約翰·高特曼(John Gottman)博士基於40年婚姻與家庭研究發展出的教養理論。核心理念是：**父母如何回應孩子的情緒，比情緒本身更重要。**

高特曼的研究發現，能夠辨識、接納、引導孩子情緒的父母，其子女在學業表現、社交能力、情緒健康、身體免疫力上都顯著優於其他孩子。這種教養方式被稱為「情緒教練」(Emotion Coaching)，將每次情緒事件視為建立親密連結和教導的機會。

**關鍵發現**：
- 經過情緒教養的孩子，壓力荷爾蒙(Cortisol)恢復速度快50%
- 學業表現平均高出10-15%
- 較少出現行為問題
- 更能調節自己的情緒
- 擁有更好的友誼品質

## 核心原則

### 1. 情緒無好壞，行為有界限 (All Feelings are OK, Not All Behaviors)

**核心概念**：
- 所有情緒都是合理且應被接納的（生氣、悲傷、嫉妒、害怕）
- 情緒是生理反應，不受意志控制
- 行為可以引導和設限
- 區分「感受」與「行為」

**對話範例**：
```
情境：孩子因弟弟弄壞積木而打人

❌ 否定情緒：
「不要生氣！生氣是不好的！」
「你當哥哥要讓弟弟，怎麼可以生氣？」

✅ 情緒教養：
「我看到你很生氣，因為弟弟弄壞你的積木。生氣是正常的。」（接納情緒）
「但是我們不可以打人。你可以說『我很生氣，請不要碰我的東西』。」（設限行為）
```

**科學依據**：
- 壓抑情緒會導致累積爆發
- 接納情緒能降低神經系統壓力反應
- 孩子學會辨識情緒是自我調節的第一步

### 2. 四種教養風格 (Four Parenting Styles)

高特曼識別出四種父母回應情緒的方式：

#### ❌ 壓抑型父母 (Dismissing Parent)
**特徵**：
- 忽視、否定、輕視孩子的情緒
- 認為負面情緒是軟弱表現
- 試圖快速「修復」情緒

**典型話語**：
- 「這有什麼好哭的？」
- 「你太敏感了！」
- 「不要想太多，去玩就好了。」

**對孩子的影響**：
- 不信任自己的感受
- 難以調節情緒
- 認為情緒是不好的

#### ❌ 不贊同型父母 (Disapproving Parent)
**特徵**：
- 批評、責備孩子的情緒
- 認為負面情緒需要糾正
- 可能使用懲罰

**典型話語**：
- 「再哭就給我去罰站！」
- 「你怎麼這麼愛生氣？」
- 「不准有這種想法！」

**對孩子的影響**：
- 羞愧感
- 情緒自我懷疑
- 可能變得叛逆或壓抑

#### ⚠️ 放任型父母 (Laissez-Faire Parent)
**特徵**：
- 接納所有情緒
- 但不提供引導或界限
- 過度同理但缺乏教導

**典型話語**：
- 「我知道你很生氣，所以打人也沒關係。」
- 「你傷心就哭吧，想哭多久都可以。」

**對孩子的影響**：
- 缺乏情緒調節策略
- 不知道如何從負面情緒中恢復
- 可能發展出行為問題

#### ✅ 情緒教練型父母 (Emotion Coaching Parent)
**特徵**：
- 意識並重視情緒
- 接納並同理情緒
- 提供引導和解決方案
- 設定行為界限

**典型話語**：
- 「我看到你很失望，因為不能去公園。」
- 「生氣是正常的，我們一起想想可以怎麼做。」
- 「你可以難過，但我們還是需要完成作業。」

**對孩子的影響**：
- 高情緒智商
- 良好自我調節能力
- 更好的問題解決能力
- 強健的親子關係

### 3. 情緒教養五步驟 (Five Steps of Emotion Coaching)

#### Step 1: 覺察情緒 (Be Aware of Emotions)
**目標**：注意到孩子的情緒，包括微妙的變化

**實踐**：
- 觀察非語言訊號（表情、姿勢、聲音）
- 注意情緒強度變化
- 辨識自己對孩子情緒的反應

**父母自我檢視**：
```
□ 我是否注意到孩子情緒低落？
□ 我能察覺孩子焦慮的早期跡象嗎？
□ 孩子的哪些情緒讓我感到不舒服？
```

**案例**：
```
情境：孩子放學回家，默默走進房間

壓抑型父母：沒注意到異狀，繼續做自己的事
情緒教練：「我注意到你今天很安靜，在學校發生什麼事嗎？」
```

#### Step 2: 將情緒視為親近與教導的機會 (Recognize Emotional Moments as Opportunities)
**目標**：重新框架情緒事件，從「麻煩」變成「機會」

**心態轉換**：
| 舊思維 | 新思維 |
|-------|-------|
| 「又在鬧脾氣了！」 | 「這是教他認識生氣的機會。」 |
| 「怎麼這麼愛哭？」 | 「這是建立連結的時刻。」 |
| 「真是麻煩！」 | 「這是他信任我的表現。」 |

**實踐**：
- 暫停手邊工作，專注當下
- 深呼吸，調節自己的情緒
- 提醒自己：「這是教導的黃金時刻」

#### Step 3: 傾聽並同理情緒 (Listen and Validate Feelings)
**目標**：讓孩子感受到被理解和接納

**同理公式**：
```
「我看到/聽到你_____(行為表現)」
「你感到_____(情緒標籤)」
「因為_____(原因)」
```

**範例**：
```
情境1：考試考不好
「我看到你把考卷揉掉了，你一定很失望，因為你這次很認真準備。」

情境2：朋友不跟他玩
「你看起來很難過，因為小明今天沒有跟你玩，對嗎？」

情境3：要收玩具
「你看起來很生氣，你想繼續玩，不想現在停下來。」
```

**注意事項**：
- ❌ 不要立刻給建議（「你應該...」）
- ❌ 不要轉移注意力（「別想了，我們去吃冰淇淋」）
- ❌ 不要比較（「你看隔壁哥哥都不會這樣」）
- ✅ 單純傾聽和反映

**身體語言**：
- 蹲下與孩子視線平行
- 眼神接觸
- 溫和的肢體接觸（如輕拍肩膀）
- 點頭表示理解

#### Step 4: 協助孩子命名情緒 (Help Children Label Emotions)
**目標**：擴展情緒詞彙，提升情緒辨識能力

**為何重要**：
- 研究顯示：能命名情緒的孩子，能更快平靜下來
- 神經科學發現：語言化情緒能降低杏仁核活化（恐懼中樞）
- 情緒詞彙豐富的孩子，更能精確表達需求

**情緒詞彙階梯**（由簡至繁）：

**基礎情緒（2-4歲）**：
- 開心、難過、生氣、害怕

**進階情緒（5-8歲）**：
- 失望、嫉妒、擔心、興奮、尷尬、驕傲

**複雜情緒（9歲以上）**：
- 焦慮、挫折、羞愧、內疚、感激、孤獨

**教導技巧**：
```
1. 繪本：《我的感覺系列》、《情緒寶盒》
2. 情緒臉譜圖：讓孩子指出感受
3. 情緒溫度計：評估情緒強度（1-10）
4. 日常生活標籤：「媽媽現在感到疲倦」
```

**對話範例**：
```
孩子：「我不想去學校！」（哭泣）
父母：「你看起來很難過，也有一點害怕。是這樣嗎？」
孩子：「嗯...」
父母：「你擔心什麼呢？」
孩子：「老師會罵我作業沒寫完...」
父母：「哦，你感到焦慮，因為作業還沒完成。」
```

#### Step 5: 設定界限並引導解決問題 (Set Limits and Problem-Solve)
**目標**：在接納情緒的同時，引導適當行為和解決方案

**兩階段執行**：

**階段一：設定行為界限**
```
「所有感受都可以，但_____(行為)不可以。」

範例：
「你可以生氣，但不可以打人。」
「你可以失望,但不可以摔東西。」
「你可以不想分享,但不可以搶玩具。」
```

**階段二：共同解決問題（五步驟）**
1. **詢問孩子的想法**
   「你覺得可以怎麼做？」

2. **評估建議**
   「這個方法可能會發生什麼事？」

3. **提供選項（必要時）**
   「你可以選擇A或B」

4. **選擇方案**
   「你決定要試試哪個方法？」

5. **事後檢討**
   「這個方法有用嗎？需要調整嗎？」

**完整對話範例**：
```
情境：孩子因弟弟弄壞積木而尖叫

父母：「你很生氣，花了一小時的積木被弄壞了。」（同理）
孩子：「我討厭他！」
父母：「你現在真的很生氣。生氣是正常的。」（接納情緒）
父母：「但是我們不可以一直尖叫，這樣大家耳朵會痛。」（設限）
父母：「你覺得怎麼做可以讓你感覺好一點？」（引導解決）
孩子：「我不知道...」
父母：「你想要重新蓋一個？還是需要先休息一下？或是我們一起跟弟弟說你的感受？」
孩子：「我想要自己蓋...」
父母：「好，那我們先確保弟弟不會再碰。要在你的房間蓋嗎？」
孩子：「好...」
父母：「等你蓋好了，我們來想想怎麼跟弟弟說『不可以碰哥哥的東西』，好嗎？」
```

### 4. 情緒銀行帳戶 (Emotional Bank Account)
**概念**：每次正向互動是「存款」，負向互動是「提款」

**健康關係比例**（高特曼研究）：
- **5:1黃金比例**：5次正向互動 : 1次負向互動
- 低於這個比例，關係品質顯著下降

**存款方式**：
1. 主動傾聽
2. 表達感謝
3. 肢體親密（擁抱、親吻）
4. 專屬一對一時間
5. 真誠道歉

**提款方式**（應避免）：
1. 批評人格（「你真是個壞孩子」）
2. 輕視（「這有什麼大不了」）
3. 防衛（「我也是為你好！」）
4. 冷戰（不理睬）

### 5. 父母的元情緒理念 (Meta-Emotion Philosophy)
**定義**：父母對情緒的信念和態度

**自我檢視問題**：
```
□ 我認為哪些情緒是「不好的」？
□ 我的原生家庭如何處理情緒？
□ 我最不舒服的情緒是什麼？（生氣？悲傷？恐懼？）
□ 我希望孩子學會如何面對情緒？
□ 我自己如何調節情緒？
```

**常見阻礙與突破**：
| 阻礙信念 | 突破思維 |
|---------|---------|
| 「男孩不應該哭」 | 「眼淚是健康的情緒釋放」 |
| 「負面情緒會影響孩子」 | 「壓抑情緒才會造成傷害」 |
| 「我需要讓孩子開心」 | 「我的角色是陪伴，不是消除所有不適」 |
| 「孩子應該自己處理」 | 「教導情緒調節需要大人引導」 |

## 實務應用

### 工具1：情緒溫度計 (Emotion Thermometer)
**適用年齡**：4歲以上

**製作方式**：
1. 畫一個溫度計（0-10刻度）
2. 0 = 很平靜，10 = 非常強烈
3. 用顏色區分（綠→黃→橙→紅）

**使用方法**：
```
父母：「你現在的生氣是幾分？」
孩子：「8分！」
父母：「哇，8分很高耶。你需要什麼可以降到5分？」
孩子：「抱抱...」
父母：「好，我們抱抱。」（擁抱）
父母：「現在呢？」
孩子：「6分了...」
父母：「進步了！還需要什麼嗎？」
```

**好處**：
- 將抽象情緒具體化
- 教導情緒強度概念
- 培養自我覺察能力

### 工具2：情緒急救箱 (Calm Down Kit)
**內容物建議**：
- **觸覺**：軟墊、絨毛玩具、黏土
- **視覺**：寧靜圖片、萬花筒、閃光瓶
- **聽覺**：音樂播放器、白噪音
- **嗅覺**：薰衣草精油、香氛袋
- **工具**：深呼吸卡、情緒圖卡

**建立步驟**：
1. 與孩子共同挑選物品
2. 裝飾盒子
3. 放在易取得的地方
4. 平時練習使用

**使用時機**：
- 情緒升高時
- 需要冷靜時
- 睡前放鬆時

### 工具3：情緒日記 (Feelings Journal)
**適用年齡**：7歲以上

**記錄內容**：
1. 今天發生什麼事？
2. 我感受到什麼情緒？
3. 我的身體有什麼感覺？
4. 我做了什麼？
5. 下次可以怎麼做？

**父母角色**：
- 不評判
- 不糾正
- 表達好奇和興趣
- 必要時引導反思

### 工具4：情緒圖卡遊戲 (Emotion Cards Game)
**製作方式**：
- 準備各種情緒臉孔圖卡
- 加上情緒詞彙

**遊戲方式**：
1. **情緒猜猜樂**：比表情，猜情緒
2. **情緒接龍**：「當我___，我會感到___」
3. **情緒故事**：抽卡編故事
4. **情緒配對**：情境與情緒配對

**教育目標**：
- 擴展情緒詞彙
- 辨識他人情緒
- 培養同理心

### 工具5：RAIN技巧（給父母）
**目的**：快速自我調節，避免情緒失控

**步驟**：
- **R**ecognize（辨識）：「我現在很生氣」
- **A**llow（允許）：「生氣是正常的」
- **I**nvestigate（探索）：「我為什麼生氣？身體什麼感覺？」
- **N**urture（滋養）：「我需要深呼吸/休息」

**實踐範例**：
```
情境：孩子第五次不收玩具，你快爆炸了

R：「我感到非常挫折和憤怒」
A：「這很正常，重複說同樣的話確實令人沮喪」
I：「我的肩膀緊繃，呼吸急促。我擔心孩子不尊重我。」
N：「我需要先離開房間，深呼吸三次，再回來處理。」

（3分鐘後，冷靜返回）
「我剛才很生氣，所以先去冷靜了一下。現在我們來談談玩具的事。」
```

## 適用情境

### 最適合的場景：
1. **日常情緒事件**：
   - 挫折、失望、生氣、害怕
   - 同儕衝突
   - 學業壓力
   - 手足競爭

2. **情緒發展階段**：
   - 2-3歲情緒爆發期
   - 學齡前情緒認知發展
   - 青春期情緒波動

3. **特殊挑戰**：
   - 高敏感孩子
   - 焦慮傾向孩子
   - 情緒調節困難孩子

### 需要調整的情境：
1. **立即危險**：先確保安全，事後再教導
2. **公共場合**：簡化步驟，事後完整處理
3. **時間緊迫**：「我們現在要出門，回來再好好談」
4. **父母情緒失控**：先自我調節，再回應孩子

### 何時尋求專業協助：
- 情緒困擾持續超過2週
- 影響日常功能（上學、睡眠、飲食）
- 出現自傷或傷人行為
- 極端焦慮或憂鬱症狀

## 常見問題與解答

### Q1: 孩子只是想操控我，假裝難過來得到想要的東西怎麼辦？
**A**: 即使孩子有目的，情緒仍是真實的。關鍵是接納情緒但設定界限。

```
孩子：「我好難過，你都不買玩具給我！」（哭泣）
父母：「你很想要那個玩具，沒買到很失望。」（接納情緒）
父母：「不過哭也不會改變我的決定，我們今天不買玩具。」（設限）
父母：「你可以難過，需要抱抱嗎？」（提供安慰，不是滿足要求）
```

### Q2: 同理了半天，孩子還是繼續哭鬧怎麼辦？
**A**: 情緒需要時間釋放，不要急著「解決」。

**重點**：
- 目標不是「停止哭泣」，而是「陪伴度過」
- 有時孩子需要「哭完」才能平靜
- 持續陪伴，不需一直說話
- 身體接觸（如抱著）比言語更有力

**實踐**：
```
父母：「我看到你很難過，我在這裡陪你。」
（靜靜陪伴，偶爾輕拍背）
（5-10分鐘後，孩子哭聲減弱）
父母：「感覺好一點了嗎？」
```

### Q3: 我很忙，沒時間每次都做情緒教養怎麼辦？
**A**: 不需要每次都完整執行，掌握核心原則即可。

**簡化版**（30秒）：
1. 命名情緒：「你很生氣」
2. 表達理解：「我知道了」
3. 簡單引導：「深呼吸，我們等下再談」

**完整版**（5-10分鐘）：
- 選擇重要時刻（強烈情緒、新情境）
- 晚上睡前是好時機（回顧一天）

### Q4: 孩子說「你不懂！」拒絕溝通怎麼辦？
**A**: 尊重孩子的界限,保持連結。

**回應方式**：
```
孩子：「你不懂啦！」（摔門）

❌ 追問：「我當然懂！你給我出來說清楚！」

✅ 尊重：
「好，我現在可能不懂。但我想理解。」
「你需要一個人靜一靜，我理解。」
「我就在客廳，準備好再來找我。」

（30分鐘後，孩子出來）
「準備好談談了嗎？我想聽聽發生什麼事。」
```

### Q5: 父母雙方教養方式不同怎麼辦？
**A**:
1. **私下討論**：不在孩子面前爭論
2. **尋求共識**：都讀同一本書，參加同一個課程
3. **尊重差異**：不同方式都可以，關鍵是一致性
4. **孩子適應力強**：他們能理解「爸爸的方式」和「媽媽的方式」不同

**建議**：
- 一起閱讀《教孩子跟情緒做朋友》
- 參加情緒教養工作坊
- 定期家庭會議討論教養議題
