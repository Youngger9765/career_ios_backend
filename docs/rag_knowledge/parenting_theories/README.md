## 🎯 理論選擇指南

### 依照教養挑戰選擇理論：

#### 行為管理問題
- **孩子不聽話、權力鬥爭** → **正向教養** (01)
  - 工具：和善而堅定、自然後果、家庭會議

- **孩子缺乏動機、過度依賴獎勵** → **自我決定論** (05)
  - 工具：三大需求、內在動機培養

#### 情緒相關問題
- **情緒失控、哭鬧、焦慮** → **情緒教養** (02)
  - 工具：情緒教養五步驟、情緒溫度計

- **分離焦慮、缺乏安全感** → **依附理論** (03)
  - 工具：安全基地、修復依附

#### 學習與發展問題
- **學習困難、期待不符** → **認知發展理論** (04)
  - 工具：發展階段檢核、ZPD鷹架

- **缺乏學習興趣** → **自我決定論** (05) + **認知發展** (04)
  - 整合應用

#### 親子關係問題
- **親子疏離、溝通困難** → **情緒教養** (02) + **依附理論** (03)
- **青春期叛逆** → **自我決定論** (05) + **認知發展** (04)

### RAG 查詢範例

**範例1：處理孩子哭鬧**
```
查詢：「孩子在超市因為不買玩具而哭鬧，該怎麼辦？」

相關理論：
1. 情緒教養 (02) → 情緒教養五步驟
2. 正向教養 (01) → 和善而堅定、邏輯後果
3. 自我決定論 (05) → 承認感受+堅定界限

整合回應：
- 步驟1：同理情緒（情緒教養）
- 步驟2：堅定界限（正向教養）
- 步驟3：提供選擇（自我決定論）
```

**範例2：孩子不寫功課**
```
查詢：「7歲孩子每天不想寫功課，總是拖延」

相關理論：
1. 自我決定論 (05) → 內在動機、自主支持
2. 認知發展 (04) → 具體運思期特徵
3. 正向教養 (01) → 慣例表、自然後果

整合應用：
- 評估：符合發展階段嗎？（認知發展）
- 動機：提供選擇、解釋意義（自我決定論）
- 結構：建立慣例、清楚期待（正向教養）
```

## 🌟 理論整合使用

### 五大理論的互補性

```
                    親子關係品質
                         ↑
                    依附理論 (03)
                  情緒教養 (02)
                         |
    行為引導 ←―― 正向教養 (01) ――→ 認知發展 (04) ――→ 學習支持
                         |
                 自我決定論 (05)
                         ↓
                    長期動機與自主
```

### 整合應用案例

**情境：5歲孩子不願上幼兒園**

**步驟1：理解發展階段**（認知發展04）
- 前運思期，分離焦慮正常
- 時間概念模糊，需要具體

**步驟2：評估依附安全**（依附理論03）
- 是否有安全依附？
- 需要建立信任感

**步驟3：處理情緒**（情緒教養02）
- 承認恐懼：「你不想去，有點害怕」
- 同理但不縱容

**步驟4：提供選擇**（自我決定論05）
- 「你想帶小熊還是小兔去學校？」
- 滿足自主需求

**步驟5：建立慣例**（正向教養01）
- 固定道別儀式
- 可預測的接送時間

## ⚠️ 使用注意事項

### 理論的限制
1. **不是萬能**：沒有單一理論能解決所有問題
2. **需要整合**：最佳實踐是整合多個理論
3. **個別差異**：需根據孩子氣質、發展調整
4. **文化脈絡**：某些概念需要文化適應

### 何時尋求專業協助
- 理論工具無效超過3個月
- 孩子出現嚴重行為/情緒問題
- 父母自身無法調節情緒
- 懷疑發展遲緩或心理疾病
- 家庭系統性問題（如父母衝突）

**建議專業**：
- 兒童心理師
- 家族治療師
- 發展評估專家
- 親職教育諮詢師

## 📚 延伸學習路徑

### 初學者（0-3個月）
1. 先讀**正向教養** (01)：最實用、立即可用
2. 再讀**情緒教養** (02)：處理日常情緒
3. 理解**依附理論** (03)：建立關係基礎

### 進階者（3-6個月）
4. 學習**認知發展** (04)：理解孩子思維
5. 深入**自我決定論** (05)：培養長期動機

### 精通者（6個月以上）
6. 整合應用：根據情境靈活運用
7. 閱讀原文書籍：深化理論理解
8. 參加工作坊：實踐演練
9. 成為認證講師：分享經驗
