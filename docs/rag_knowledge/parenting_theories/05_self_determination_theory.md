# 自我決定論 (Self-Determination Theory)

## 理論概述

自我決定論(Self-Determination Theory, SDT)是由心理學家愛德華·德西(Edward Deci)和理查·萊恩(Richard Ryan)發展的動機理論。核心主張是：**人類有三個基本心理需求——自主性、勝任感、關聯感。當這三個需求被滿足時,人們會展現內在動機、最佳表現和心理健康。**

SDT對親職教育的革命性貢獻在於：它挑戰了傳統「胡蘿蔔加棍子」的教養方式，證明外在獎懲會破壞內在動機，而支持自主性的教養能培養出更有動機、更快樂、更成功的孩子。

**關鍵洞見**：
- 內在動機比外在獎勵更持久、更深刻
- 控制型教養破壞自主性，長期有害
- 支持自主≠放任，仍需提供結構與界限
- 滿足三大需求的孩子有更好的學業、社交、心理健康

  - 著有《Why We Do What We Do》（為什麼我們這樣做）

```
實驗：大學生玩有趣的拼圖遊戲

第二組：無獎勵

結果（休息時間自由活動）：
第一組：不再玩拼圖（失去興趣）
第二組：繼續玩拼圖（保持興趣）

結論：外在獎勵破壞內在動機
```

## 核心原則

### 1. 三大基本心理需求 (Three Basic Psychological Needs)

#### 需求一：自主性 (Autonomy)

**定義**：感受到自己是行為的主體，能自由選擇和決定。

**不是什麼**：
- ❌ 不是獨立(Independence)：不需要他人
- ❌ 不是放任(Permissiveness)：想做什麼就做什麼
- ❌ 不是反抗(Defiance)：拒絕所有規則

**是什麼**：
- ✅ 感到有選擇權
- ✅ 行為符合真實自我
- ✅ 理解並認同行為的意義
- ✅ 感受到被尊重

**自主性光譜**：
```
低自主 ←――――――――――――――――――――→ 高自主
外在調節     內攝調節     認同調節     整合調節     內在動機

外在調節（完全外控）：
「不寫作業就處罰」
→ 為了避免懲罰而做

內攝調節（部分內化）：
「我應該要寫作業,不然會有罪惡感」
→ 為了避免羞愧、焦慮

認同調節（認同價值）：
「寫作業對學習很重要」
→ 理解意義,但不一定享受

整合調節（完全內化）：
「寫作業幫助我達成目標,符合我的價值觀」
→ 與自我一致

內在動機（最高自主）：
「我喜歡學習,寫作業很有趣」
→ 純粹享受過程
```

**支持自主 vs. 控制型教養**：
| 自主支持 | 控制型 |
|---------|--------|
| 提供選擇與彈性 | 強迫、命令 |
| 解釋原因與意義 | 「因為我說了算」 |
| 承認孩子感受 | 否定、壓抑情緒 |
| 鼓勵主動性 | 過度指導 |
| 尊重觀點 | 忽視想法 |
| 提供結構（清楚期待） | 混亂或過度控制 |

**對話範例**：
```
情境：孩子不想寫功課

❌ 控制型：
「現在立刻去寫！不寫就不准看電視！」
（威脅、外在控制）

⚠️ 部分自主（內攝調節）：
「你應該要寫功課,不然老師會罵你,多丟臉！」
（誘發羞愧感,仍是控制）

✅ 自主支持：
「我知道你現在不想寫功課。」（承認感受）
「功課需要在睡前完成。」（清楚期待）
「你想現在寫,還是吃完晚餐後寫？」（提供選擇）
「寫功課可以幫助你理解今天學的內容。」（解釋意義）
```

**支持自主的技巧**：
1. **提供選擇**（在合理範圍內）
   - 「你要先寫數學還是國語？」
   - 「你要在書桌寫還是餐桌？」

2. **解釋原因**（不是命令）
   - 「我們需要準時出門,因為...」
   - 「刷牙保護牙齒,避免蛀牙」

3. **承認感受與觀點**
   - 「我知道你不想離開公園」
   - 「你希望繼續玩,對吧？」

4. **最小必要壓力**
   - 「如果可以,我希望你...」
   - 「我建議你...，你覺得呢？」

5. **鼓勵主動性**
   - 「你有什麼想法？」
   - 「你覺得怎麼做比較好？」

#### 需求二：勝任感 (Competence)

**定義**：感受到自己有能力、能有效與環境互動、能達成目標。

**不是什麼**：
- ❌ 不是完美主義：每件事都要做到100分
- ❌ 不是與他人比較：比別人強
- ❌ 不是天賦：天生聰明

**是什麼**：
- ✅ 感到有效能：「我做得到」
- ✅ 看見進步：「我比之前更好了」
- ✅ 掌握挑戰：「這很難,但我能學會」
- ✅ 獲得真誠回饋：知道自己做得如何

**最佳挑戰理論**（維高斯基ZPD的SDT版）：
```
太簡單（無聊）←→【最佳挑戰】←→ 太困難（焦慮）
               ↑
            培養勝任感
```

**支持勝任感 vs. 破壞勝任感**：
| 支持勝任感 | 破壞勝任感 |
|-----------|-----------|
| 提供適當挑戰 | 任務太難或太簡單 |
| 真誠、具體回饋 | 空泛讚美或批評 |
| 關注進步 | 只看結果 |
| 允許錯誤 | 懲罰失敗 |
| 教導策略 | 不提供協助 |
| 鼓勵努力 | 只稱讚天賦 |

**對話範例**：
```
情境：孩子學騎腳踏車,一直跌倒

❌ 破壞勝任感：
「你怎麼這麼笨！這麼簡單都學不會！」
「算了,你大概沒天分,不要學了。」
（羞辱、放棄）

⚠️ 空泛讚美（無效）：
「你好棒棒！」（孩子明明一直跌倒,讚美不真誠）

✅ 支持勝任感：
「騎腳踏車確實需要練習,你已經進步了！」（承認難度）
「你看,這次你撐了5秒,比剛才的2秒進步！」（具體回饋）
「我們來想想,怎麼保持平衡？」（教導策略）
「跌倒是正常的,我學的時候也跌了很多次。」（正常化錯誤）
「要不要再試一次？我扶著你。」（支持挑戰）
```

**培養勝任感的技巧**：
1. **提供結構化學習**
   - 分解任務（小步驟）
   - 清楚說明期待
   - 提供示範與練習機會

2. **給予有效回饋**
   - **真誠**：基於真實表現
   - **具體**：「你把玩具都分類好了」>「你好棒」
   - **過程導向**：「你很努力思考」>「你很聰明」
   - **建設性**：「下次可以試試...」

3. **設定適當挑戰**
   - 觀察孩子能力（ZPD）
   - 逐漸增加難度
   - 提供鷹架支持

4. **正常化失敗**
   - 「錯誤是學習的一部分」
   - 分享自己的失敗經驗
   - 關注「從錯誤學到什麼」

5. **慶祝進步（不只是成就）**
   - 「你現在能自己穿襪子了！記得上個月還需要幫忙。」
   - 成長日記、進步牆

#### 需求三：關聯感 (Relatedness)

**定義**：感受到與他人有連結、被關心、屬於某個群體。

**不是什麼**：
- ❌ 不是依賴：無法獨立
- ❌ 不是取悅他人：為了被喜歡而犧牲自我
- ❌ 不是從眾：放棄自己的想法

**是什麼**：
- ✅ 感到被愛、被接納
- ✅ 能信任與依靠他人
- ✅ 能關心他人
- ✅ 歸屬感

**親子關聯感的表現**：
- 孩子願意分享感受
- 尋求安慰時會來找父母
- 享受與父母共處時光
- 感受到被理解
- 知道即使犯錯也會被愛

**支持關聯感 vs. 破壞關聯感**：
| 支持關聯感 | 破壞關聯感 |
|-----------|-----------|
| 溫暖、關心 | 冷漠、疏離 |
| 專注傾聽 | 分心、忽視 |
| 情緒回應 | 否定情緒 |
| 品質時間 | 忙碌、缺席 |
| 無條件接納 | 有條件的愛 |
| 同理理解 | 批判、羞辱 |

**有條件的愛（破壞關聯感）**：
```
❌ 表現好才愛你：
「你考100分,媽媽才愛你」
「你這麼不乖,我不要你了」

心理影響：
- 孩子學會壓抑真實自我
- 為了獲得愛而表現
- 發展出「假自我」(False Self)
- 長期焦慮、低自尊
```

**無條件的愛（支持關聯感）**：
```
✅ 分離行為與人格：
「你打妹妹這個行為不對,我們需要改變。但我永遠愛你。」
「我不喜歡你剛才的行為,但我愛你這個人。」

✅ 一致的愛：
無論表現好壞、聽話與否,父母的愛不改變
```

**對話範例**：
```
情境：孩子考試考差了

❌ 有條件的愛（破壞關聯感）：
「你考這麼爛,真讓我失望！」
「你看看隔壁小明,人家都考100分！」
「你再這樣,我就不愛你了！」

✅ 無條件接納（支持關聯感）：
「考試成績不理想,你一定也很難過。」（同理）
「我們一起看看哪裡需要加強。」（支持）
「無論成績如何,我都愛你。我關心的是你有沒有盡力,有沒有學到東西。」（無條件）
「你需要什麼幫助嗎？」（關心）
```

**培養關聯感的技巧**：
1. **品質時間**
   - 每天15-30分鐘專屬一對一時間
   - 全心陪伴（放下手機）
   - 做孩子想做的事

2. **主動傾聽**
   - 眼神接觸
   - 反映感受：「你聽起來很...」
   - 不打斷、不評價
   - 好奇提問

3. **情緒回應**
   - 接納所有情緒
   - 同理但不縱容行為
   - 「我在這裡陪你」

4. **日常連結**
   - 家庭儀式（晚餐、睡前故事）
   - 身體接觸（擁抱、親吻）
   - 表達愛：「我愛你」
   - 留小紙條

5. **展現興趣**
   - 詢問孩子的世界（朋友、興趣）
   - 參與孩子的活動
   - 記得重要事件

### 2. 內在動機 vs. 外在動機

**內在動機** (Intrinsic Motivation)：
- 為了活動本身的樂趣而做
- 「我喜歡閱讀,閱讀很有趣」
- 特徵：專注、創造力、持久性、深度學習

**外在動機** (Extrinsic Motivation)：
- 為了外在結果而做
- 「我讀書是為了得到獎勵/避免懲罰」
- 特徵：表面學習、壓力、結果導向

**關鍵發現：外在獎勵的過度使用效應**
```
德西1971實驗 + 後續百篇研究證實：

原本有興趣的活動 + 外在獎勵 = 興趣下降

機制：
1. 改變歸因（「我做這個是為了獎勵」而非「我喜歡」）
2. 降低自主感（「我是被控制的」）
3. 成為交易（「沒獎勵就不做」）
```

**獎勵的陷阱**：
```
情境：孩子原本喜歡畫畫

父母開始給獎勵：
「畫一張畫就給你10元！」

短期：孩子畫更多（為了錢）
長期：
- 不給錢就不畫
- 畫畫變成工作,不再享受
- 只畫最快完成的（品質下降）
- 內在動機被破壞

停止獎勵後：
畫畫頻率低於獎勵前
（因為內在動機已被破壞）
```

**什麼時候外在動機可接受？**
1. **本來就無趣的任務**
   - 刷牙（健康必要但無趣）
   - 某些家務
   - 例行公事

2. **啟動動機**（短期使用）
   - 孩子完全沒興趣,用獎勵啟動
   - **但要逐漸淡出獎勵,轉向內在動機**

3. **訊息性獎勵**（不是控制性）
   - 獎勵傳達「你做得好」（勝任感）
   - 而非「你被控制」

**獎勵類型對照**：
| 控制性獎勵（有害） | 訊息性獎勵（較無害） |
|----------------|------------------|
| 預告、預期的獎勵 | 意外、不預期的獎勵 |
| 「如果你...就給你...」 | 「哇,你做得真好,這是給你的驚喜」 |
| 標準化（做X得Y） | 個人化（肯定努力） |
| 破壞內在動機 | 較少破壞 |

**實務建議**：
```
❌ 避免：
- 「如果你讀完這本書,我給你100元」（本來可能喜歡閱讀）
- 「寫完功課就給你玩手機」（功課變成交易）
- 貼紙換獎品（控制系統）

✅ 改用：
- 支持內在動機：「這本書很有趣,你可能會喜歡」
- 提供選擇：「你想讀哪本？」
- 關注樂趣：「最喜歡哪個部分？」
- 訊息性回饋：「你很專注讀了一小時,看來你很投入這個故事」
```

### 3. 自主支持教養的四大支柱

#### 支柱一：提供選擇與彈性

**原則**：在合理界限內,最大化孩子的選擇權。

**實踐技巧**：
```
年齡適應：
2-4歲：兩個選擇
「你要穿紅色還是藍色上衣？」

5-8歲：有限選擇
「你要先寫功課、洗澡還是吃點心？」

9-12歲：更多自主
「你今晚要怎麼安排時間？9點前要上床。」

13歲以上：協商
「我們一起討論你的門禁時間,你覺得合理的是幾點？」
```

**界限內的自由**：
```
✅ 好的選擇：
- 兩個選項你都能接受
- 符合發展階段
- 有清楚界限

範例：
「我們要出門了,你要現在穿鞋,還是在車上穿？」
（兩個都OK,但不包括「不穿鞋」）

❌ 假選擇（控制型）：
「你要乖乖吃飯,還是要被處罰？」
（這不是真的選擇）
```

#### 支柱二：提供理由與解釋

**原則**：幫助孩子理解規則的意義,而非盲目服從。

**發展階段調整**：
```
幼兒期（簡單因果）：
「我們要牽手過馬路,因為車子很危險。」

學齡期（邏輯推理）：
「如果你晚睡,明天會很累,上課會不專心。」

青少年期（價值討論）：
「我們設定門禁,是因為我們重視你的安全。但我們可以討論合理的時間。」
```

**「因為我說了算」的替代**：
```
❌ 權威命令：
孩子：「為什麼要收玩具？」
父母：「因為我說的！不要問那麼多！」

✅ 提供理由：
父母：「收玩具有幾個原因：」
1. 「我們明天才能快速找到想玩的」（實用性）
2. 「家裡整齊,大家走路才不會踩到」（安全）
3. 「這是我們家的規則,大家都要遵守」（公平）
「你能理解嗎？」
```

#### 支柱三：承認與接納感受

**原則**：允許所有情緒,設限行為。

**步驟**：
```
1. 辨識情緒：「你看起來很生氣」
2. 同理：「我理解你很失望」
3. 接納：「生氣是正常的」
4. 設限（如需要）：「但我們不可以打人」
5. 引導：「你可以說『我很生氣』或去冷靜角」
```

**承認感受≠同意要求**：
```
情境：孩子想要買玩具,被拒絕後大哭

✅ 承認感受+堅定界限：
「我知道你很想要那個玩具,沒買到很失望。」（承認）
「但我們今天不買玩具。」（界限）
「你可以難過,我陪你。」（接納情緒）

❌ 否定感受：
「這有什麼好哭的！不准哭！」

❌ 因感受而妥協：
「好啦好啦,不要哭了,買給你就是了」（破壞界限）
```

#### 支柱四：最小必要壓力

**原則**：用最少的控制達成目標。

**語言技巧**：
```
控制程度排序（由高到低）：

高控制 →
「你現在立刻去刷牙！」（命令）
「你應該去刷牙！」（義務）
「該刷牙了。」（陳述事實）
「你想現在刷牙,還是5分鐘後？」（選擇）
「記得睡前要刷牙哦。」（提醒）
「你覺得現在該做什麼了？」（引導思考）
← 低控制

建議：多使用低控制語言
```

**「邀請」而非「命令」**：
```
❌ 命令：「把房間收拾乾淨！」
✅ 邀請：「你可以幫忙把房間整理一下嗎？」

❌ 命令：「坐好吃飯！」
✅ 說明期待：「吃飯時我們坐在椅子上」

❌ 威脅：「再不睡覺就處罰你！」
✅ 最小壓力：「睡覺時間到了,你要自己走去房間,還是我陪你去？」
```

### 4. 動機的連續體 (Motivation Continuum)

SDT提出動機不是「有或無」,而是連續體：

```
無動機 ←―――――――――――――――――――――――→ 內在動機
  ↓         ↓         ↓         ↓         ↓
無動機   外在調節   內攝調節   認同調節   整合調節   內在動機

完全不想做 → 為避免懲罰 → 為避免羞愧 → 認同價值 → 與自我一致 → 純粹享受
```

**教養目標**：
- ❌ 不是所有事都要達到內在動機（刷牙很難純粹享受）
- ✅ 盡可能往右移動（更內化的調節）
- ✅ 至少達到「認同調節」（理解意義）

**促進內化的技巧**：
```
從外在調節 → 內攝調節：
「不寫作業會被罵」→「我應該寫作業」
技巧：減少外在懲罰,強調內在標準

從內攝調節 → 認同調節：
「應該寫」→「寫作業對學習有幫助」
技巧：討論價值與意義

從認同調節 → 整合調節：
「對學習有幫助」→「學習是我重視的,符合我的目標」
技巧：連結到孩子的價值觀、目標、認同

理想狀態：
「我本來就想學這個」（內在動機）
```

## 實務應用

### 工具1：三大需求檢核表

**評估孩子的心理需求是否被滿足**：

**自主性檢核**：
```
□ 我提供孩子年齡適當的選擇
□ 我解釋規則背後的原因
□ 我尊重孩子的想法與觀點
□ 我避免過度控制
□ 我鼓勵孩子表達意見
□ 我承認孩子的感受

分數低？→ 增加選擇、解釋、尊重
```

**勝任感檢核**：
```
□ 我提供適當挑戰（不太難不太簡單）
□ 我給予具體、真誠的回饋
□ 我關注進步而非完美
□ 我允許孩子犯錯
□ 我教導策略而非批評
□ 我慶祝努力與成長

分數低？→ 調整挑戰、改善回饋、正常化錯誤
```

**關聯感檢核**：
```
□ 我每天有品質時間與孩子相處
□ 我專注傾聽孩子說話
□ 我回應孩子的情緒需求
□ 我表達無條件的愛
□ 我對孩子的興趣展現好奇
□ 我提供溫暖與支持

分數低？→ 增加連結時間、改善情緒回應
```

### 工具2：從獎懲轉向內在動機

**逐步淡出外在獎勵計畫**：

**階段1：評估現況**
```
列出所有使用獎勵/懲罰的行為：
- 寫功課 → 貼紙獎勵
- 收玩具 → 給零用錢
- 練琴 → 可以看電視
- 不吵鬧 → 買玩具
```

**階段2：分類**
```
A. 本來有興趣的（內在動機被破壞）：
   如畫畫、閱讀 → 優先停止獎勵

B. 中性任務（可培養內在動機）：
   如家務 → 改用關聯感、勝任感

C. 本來就討厭的（難以內在動機）：
   如刷牙 → 可暫時保留獎勵,但要淡出
```

**階段3：替代策略**
```
停止：「寫完功課給你貼紙」

替代策略（滿足三大需求）：
1. 自主性：
   「你要先寫哪一科？」
   「你想在哪裡寫？」

2. 勝任感：
   「你這題用了不同方法,很有創意！」
   「你比上週更快理解了。」

3. 關聯感：
   「需要幫忙隨時說。」
   「寫完我們一起檢查？」

4. 內化意義：
   「寫功課幫助你理解課堂內容。」
   「你覺得寫功課對你有什麼幫助？」
```

**階段4：監測與調整**
```
觀察：
- 孩子的動機有改變嗎？
- 是否更主動？還是更被動？
- 情緒如何？（快樂vs.壓力）

調整：
- 如果動機下降,增加支持（鷹架）
- 如果過度壓力,降低期待
- 持續強化三大需求
```

### 工具3：自主支持對話腳本

**高頻情境的自主支持回應**：

**情境1：孩子抗拒任務**
```
孩子：「我不想寫功課！」

❌ 控制：「不寫就不准玩！」

✅ 自主支持：
1. 承認感受：「我知道你現在不想寫。」
2. 同理：「功課確實有時候很無聊。」
3. 提供選擇：「你想現在寫,還是吃完點心後寫？」
4. 解釋原因：「但功課需要在睡前完成,這樣明天才能交給老師。」
5. 邀請合作：「我們一起想想怎麼讓寫功課不那麼痛苦？」
```

**情境2：孩子想要某物被拒絕**
```
孩子：「我要買那個玩具！」
父母：「今天不買。」
孩子：「為什麼！我就要！」（哭鬧）

✅ 自主支持+堅定界限：
1. 承認慾望：「你真的很想要那個玩具,對吧？」
2. 同理情緒：「沒買到很失望,我理解。」
3. 說明界限：「但我們今天不買玩具。」
4. 提供替代：「你可以把它加到生日願望清單。」
5. 允許情緒：「你可以難過,我陪你。」
6. 堅定：「即使你哭,我的決定也不會改變。」
```

**情境3：孩子犯錯**
```
孩子：打翻牛奶

✅ 支持勝任感+自主：
1. 保持平靜：「牛奶打翻了。」（陳述事實,不責備）
2. 正常化：「這是會發生的事。」
3. 問題解決：「你覺得我們該怎麼處理？」（自主）
4. 提供選擇：「你要拿抹布,還是拿拖把？」
5. 共同處理：「我們一起清理。」
6. 學習機會：「下次可以把杯子放桌子中間,比較不會打翻。」
```

### 工具4：內在動機培養活動

**設計支持三大需求的日常活動**：

**家務參與**：
```
傳統（外在動機）：
「收拾房間就給你10元」

SDT方式：
1. 自主性：「你想負責哪項家務？掃地、摺衣服還是擦桌子？」
2. 勝任感：「你把衣服摺得很整齊,襪子都配對好了！」
3. 關聯感：「謝謝你幫忙,讓我們家更整潔,大家都能享受。」
4. 意義：「每個家庭成員都有責任,這樣家才能運作。」
```

**學習活動**：
```
傳統（外在動機）：
「考100分就買玩具」

SDT方式：
1. 自主性：
   - 選擇學習方式：「你想看影片學,還是看書學？」
   - 選擇進度：「你今天想完成哪些部分？」

2. 勝任感：
   - 適當挑戰：提供ZPD內的任務
   - 過程回饋：「你用了圖像幫助記憶,很有策略！」
   - 進步紀錄：「你現在10分鐘能寫完,上週要20分鐘。」

3. 關聯感：
   - 共同學習：「我們一起研究這個問題」
   - 分享興趣：「你最喜歡哪個部分？」
   - 情緒支持：「這很難,但我相信你」

4. 內在興趣：
   - 連結好奇心：「你想知道為什麼嗎？」
   - 探索樂趣：「這個實驗很有趣！」
```

**遊戲時間**：
```
自由玩耍（最高自主）：
- 孩子選擇玩什麼（自主性）
- 嘗試挑戰（勝任感）
- 與父母/同儕互動（關聯感）
- 純粹享受（內在動機）

父母角色：
- 觀察不干預（除非安全問題）
- 展現興趣：「你在蓋什麼？」
- 提供材料
- 參與（如果被邀請）
```

### 工具5：青少年自主支持策略

**青春期特殊挑戰：自主需求爆發**

**理解青少年**：
```
發展任務：建立獨立認同
自主需求：極度渴望自主,抗拒控制
父母困境：如何給自由又保護安全？
```

**自主支持策略**：
```
1. 協商界限（不是單方面規定）
父母：「我們來討論門禁時間。」
「我關心的是你的安全。你覺得合理的時間是？」
青少年：「12點？」
父母：「我擔心太晚。我們能不能折衷10點半？」
「如果你能準時回來、保持聯絡,我們之後可以調整。」

2. 提供隱私（在安全範圍內）
✅ 尊重：敲門再進房間
✅ 信任：不偷看日記
✅ 界限：「我尊重你的隱私,但如果我擔心安全,可能需要介入。」

3. 重視觀點
✅「你對這件事的看法是什麼？」
✅「我可能不同意,但我想理解你的想法。」
❌「你還小,不懂」

4. 允許安全範圍內的錯誤
✅ 讓青少年承擔後果
✅ 不拯救所有問題
✅「這是你的決定,我尊重。你準備承擔後果嗎？」

5. 保持連結
✅ 即使給自主,仍保持情感連結
✅「我知道你需要空間,但我隨時在這裡。」
✅ 家庭儀式（晚餐、週末活動）
```

## 常見問題與解答

### Q1: 自主支持是不是就是放任？不管孩子？
**A**: 不是！自主支持≠放任。

**關鍵區別**：
```
放任：
- 沒有界限
- 沒有引導
- 孩子想做什麼就做什麼

自主支持：
- 有清楚界限
- 在界限內提供選擇
- 解釋原因、引導思考
- 結構+自由
```

**實例**：
```
情境：孩子想熬夜

放任：「隨便你,想睡再睡。」
控制：「9點給我上床!不准討價還價!」
自主支持：
  「學校日需要充足睡眠,9點要上床。」（界限）
  「但你可以選擇現在洗澡,還是8點半洗。」（選擇）
  「也可以選擇看書還是聽音樂入睡。」（自主）
```

### Q2: 完全不用獎勵懲罰,孩子會聽話嗎？
**A**: SDT不是說「絕對不用」,而是「謹慎使用,不依賴」。

**實務平衡**：
```
優先使用（內在動機策略）：
1. 滿足三大需求
2. 解釋意義
3. 提供選擇
4. 培養興趣

必要時使用（外在策略）：
1. 本來就無趣的任務（刷牙）
2. 啟動動機（暫時使用,逐漸淡出）
3. 訊息性獎勵（肯定而非控制）

絕對避免：
1. 有條件的愛
2. 羞辱、威脅
3. 過度控制
```

### Q3: 我的孩子已經習慣獎勵了,現在停止會不會反彈？
**A**: 可能會有過渡期,需要策略性淡出。

**淡出計畫**：
```
步驟1：解釋改變
「我們想試試新方法。以前用貼紙,但現在我們想幫助你找到自己想做的原因。」

步驟2：逐漸減少頻率
每天獎勵 → 每週獎勵 → 偶爾意外獎勵 → 停止

步驟3：同時增加內在支持
- 更多選擇
- 更多回饋（非物質）
- 討論意義

步驟4：預期過渡反應
「我不想做,因為沒有貼紙！」
回應：「我知道改變不容易。我們來想想除了貼紙,做這件事還有什麼好處？」

步驟5：堅持與調整
- 可能需要1-3個月
- 觀察動機變化
- 必要時調整策略
```

### Q4: 華人文化重視服從,SDT適用嗎？
**A**: 研究顯示三大需求是跨文化普世的,但表達方式可能不同。

**文化調適**：
```
自主性的文化差異：
西方：個人選擇、獨立
東方：關係中的自主、和諧

華人文化中的自主支持：
- 強調「為家庭」的選擇（關聯中的自主）
- 「孝順」可以內化（認同調節）
- 「我們一起決定」（集體自主）

範例：
西方自主：「你想學什麼才藝？」
華人自主：「這對你未來有幫助,也是家裡期待,但你可以選擇學鋼琴或小提琴。」
（關聯+自主）
```

**台灣SDT研究（陳舜文等）發現**：
- 三大需求在台灣同樣預測幸福感
- 自主支持的華人父母,孩子表現更好
- 關鍵：解釋意義時連結家庭價值

### Q5: 我試著給選擇,但孩子都選我不能接受的,怎麼辦？
**A**: 選擇需要在「合理界限」內。

**設計選擇**：
```
❌ 開放式（可能失控）：
「你想做什麼？」
孩子：「我要玩一整天！」

✅ 界限內選擇：
「功課需要完成。你要現在寫,還是吃完晚餐寫？」
（兩個選項都可接受）

如果孩子說：「我都不要！」
回應：
「我知道你都不想做。」（同理）
「但功課是必須的。」（界限）
「如果你不選,我會幫你決定。」（後果）
「你想自己選,還是我來選？」（重新框架選擇）
```
